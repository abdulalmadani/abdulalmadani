## 🔭 Featured Projects

**[ai-soc-honeypot](https://github.com/abdulalmadani/ai-soc-honeypot)**: a detection-engineered Azure honeypot feeding Microsoft Sentinel, with a Python AI agent that triages every alert.
- 🤖 **AI triage + deterministic guardrails**: deterministic rules check the agent's output and override the model when it's wrong, so a bad call is caught and corrected rather than shipped.
- 🧪 **Built to be measured**: ships an evaluation harness for systematic failure analysis, not just a one-off demo.

**[ast10-scanner](https://github.com/abdulalmadani/ast10-scanner)**: an automated AI-agent security auditor that maps agent skill manifests to the OWASP Agentic Skills Top 10.
- 🧩 **Hybrid two-layer engine**: deterministic rules paired with five specialized LLM agents (Python · CrewAI · FastAPI backend, React front end).
- 🎯 **Adversarially validated**: tested against adversarial inputs, not just happy-path demos.
